Galerie realizata pe acelasi principiu cu Control TOGGLE

Se utilizeaza o abordare pur HTML/CSS (fara JavaScript).

Controlul este de fapt format din label-urile aferente la doua input-uri de tip "radio" cu acelasi name. Se pot adauga oricate imagini.

Se recomanda utilizarea de imagini cu aceeasi dimensiune (atat pentru thumbnail cat si pentru imaginea mare). In exemplu au fost utilizate imagini cu dimensiuni diferite.

Selectorul CSS utilizat este de forma: .galerie input[type="radio"]:checked+label
